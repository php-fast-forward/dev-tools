Getting Started
===============

This section explains how to install the package, what it changes inside a
consumer repository, and how to reach the first successful run without already
knowing the internal architecture.

By the end of this section you will know how to:

- install the Composer plugin safely;
- run the generated commands through Composer or directly through the binary;
- prepare the minimum ``docs/`` structure required by the documentation
  pipeline;
- understand which files ``dev-tools:sync`` creates or updates, including
  packaged skills under ``.agents/skills`` and packaged project agents under
  ``.agents/agents``.
