# Support

## How to Get Help

If you need help using FastForward DevTools, start with these resources:

- Documentation: <https://php-fast-forward.github.io/dev-tools/>
- Issue tracker: <https://github.com/php-fast-forward/dev-tools/issues>
- Wiki: <https://github.com/php-fast-forward/dev-tools/wiki>

## Which Channel to Use

- **Usage questions**: open a GitHub issue with enough project context to
  explain what you are trying to do.
- **Bug reports**: use the bug report issue form.
- **Feature requests**: use the feature request issue form.
- **Security reports**: follow [`SECURITY.md`](SECURITY.md) and report them
  privately.

## Before Opening an Issue

Please check:

- the published documentation
- existing open and closed issues
- whether the behavior can be reproduced with the current `main` branch or
  latest published release
