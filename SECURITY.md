# Security Policy

## Supported Versions

FastForward DevTools is maintained on the current `main` line.

| Version | Supported |
|---------|-----------|
| `main`  | Yes |
| Older releases | No |

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Instead, report vulnerabilities privately to:

- Email: [github@mentordosnerds.com](mailto:github@mentordosnerds.com)

When possible, include:

- a clear description of the issue
- affected versions or commit ranges
- reproduction steps or a proof of concept
- expected impact
- any suggested mitigation

## Response Expectations

We aim to:

- acknowledge the report within 5 business days
- keep the reporter informed as triage progresses
- coordinate a fix and disclosure timeline when the report is confirmed

Please allow time for investigation before public disclosure.
